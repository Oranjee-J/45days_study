### 第 12 题

请解释"数据泄露（Data Leakage）"是什么，并举一个具体例子说明它如何发生以及为什么危险。
数据泄露就是测试集里的数据有一部分混合到训练集和验证集里去了。
比如可能在数据集里有几条是重复数据，但是在数据处理阶段没有进行去重，就开始随机划分，有几条被划分到训练集，有几条被划分到测试集。
这会导致模型尽管过拟合或者欠拟合了，但可能表现结果很好，但是表现结果好是因为测试集被污染了，其中有训练集里的数据。
我们评估模型是希望评估模型有没有学到数据的规律，因此要用对模型而言的全新数据去测试。
真正的表现并没有通过测试阶段正式评估出来，那么在进入使用时，我们所得到的结果误差将会非常大。

### 第 13 题

请对比说明：过拟合和欠拟合分别表现为什么？各自的常见原因是什么？有哪些缓解方法？
过拟合表现为模型拟合训练集的时候loss非常小并在不断下降，而模型拟合验证集时loss先降低后升高。
常见原因是模型复杂度过高，层数过多，或者特征太少。
缓解方法是 1.降低模型复杂度，减少神经网络层数； 2. 增加特征 3. 添加正则化或早停，当验证集loss下降到一定范围内时就停止循环 4. 交叉验证

欠拟合表现为模型拟合训练集和测试集时loss都非常大，模型表现不好。
常见原因是模型复杂度过低，层数过少，模型不能准确识别数据里的规律。
缓解方法是 1.增加模型复杂度，增加神经网络层数； 2. 减少正则化（比如减小系数），去除早停     3. 在对数据处理的时候对某些数据进行平方、取对数等处理，来帮助模型识别非线性误差。

### 第 15 题

使用 sklearn 对鸢尾花数据集进行 **5 折交叉验证**：

1. 使用决策树分类器 `DecisionTreeClassifier(random_state=42)`
2. 使用 `cross_val_score` 进行 5 折交叉验证
3. 打印每折的准确率和平均准确率

**这题我不会**

```python
from sklearn.datasets import load_iris
from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import cross_val_score

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# 创建模型
clf = DecisionTreeClassifier(random_state=42)

# 5 折交叉验证
scores = cross_val_score(clf, X, y, cv=5, scoring='accuracy')

# 打印结果
print(f"每折准确率：{scores}")
print(f"平均准确率：{scores.mean():.4f}")
print(f"标准差：{scores.std():.4f}")
```
