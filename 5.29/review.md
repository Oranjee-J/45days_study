## 线性回归
线性回归 (Linear Regression) 是一种用于预测连续值的最基本的机器学习算法，它假设目标变量 y 和特征变量 x 之间存在线性关系，并试图找到一条最佳拟合直线来描述这种关系。

线性回归常用的误差函数是均方误差 (MSE)：
MSE = 1/n * Σ(y_i - y_pred_i)^2



手动梯度下降的方法：
```python
for i in range(n_iterations):
    y_pred = w * x + b
    dw = -(2/len(x)) * np.sum(x * (y - y_pred))
    db = -(2/len(x)) * np.sum(y - y_pred)
    w = w - learning_rate * dw
    b = b - learning_rate * dbs
```

为什么输出概率？

逻辑回归是一种用于二分类或多分类问题的算法。通过逻辑函数将线性回归的输出映射到0-1之间，从而预测某个时间发生的概率。
逻辑回归主要分成两步：
第一步：计算一个线性结果
第二部：Sigmoid映射
得到某一类别的概率

逻辑回归用交叉熵损失作为损失函数，因为：
1. 更符合概率模型
2. 梯度更稳定
3. 对错误高置信度惩罚很大

逻辑函数输出概率的原因：
1. 没办法确定两个类别差距多大，接不接近边界（因为可能出现51%认为是猫的情况，如果不输出概率就体现不出来）
2. 概率能表达置信度 
3. 概率能用于损失函数优化，概率为梯度下降提供了可能

## 损失函数
损失函数用来量化模型预测值与真实值之间的差异

均方误差 - 适用于回归问题（预测连续值，如房价、温度）
均方误差计算的是所有样本的预测值与真实值之差的平方的平均值。
均方误差公式：
 MSE = (1/n) * Σ(真实值ᵢ - 预测值ᵢ)²

交叉熵损失 - 适用于分类问题
交叉熵衡量的是模型预测的概率分布与真实的概率分布之间的差异
二分类公式：
Log Loss = - (1/n) * Σ [真实值ᵢ * log(预测概率ᵢ) + (1 - 真实值ᵢ) * log(1 - 预测概率ᵢ)]

## 正则化
正则化（Regularization）是机器学习和统计学中用来防止过拟合的一类方法。

Loss=误差+λ×惩罚项

最经典的正则化方法是在损失函数中直接添加一个基于模型权重参数的惩罚项。根据惩罚项的计算方式不同，主要分为 L1 和 L2 正则化。
L1正则化：
$$L(\mathbf{w})=\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat y_i)^2+\lambda\sum_{j=1}^{m}|w_j|$$
- 惩罚参数绝对值
- 会让某些参数直接变成 0    wj*​xj​=0，参数变成0，与之相乘的该特征也相当于没有参与过这个模型
- 可以做“特征选择”  筛选重要特征

L2正则化：
$$L(\mathbf{w})=\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat y_i)^2+\lambda\sum_{j=1}^{m}w_j^2$$
- 惩罚参数平方
- 参数会变小，但通常不会变成0   让模型更平滑
- 最常用
