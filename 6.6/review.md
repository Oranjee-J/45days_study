## batchnorm
batchnorm指batch normalization，也就是批量归一化，目的是解决深度神经网络训练中的内部协变量偏移问题。

```python
def batch_norm(x, gamma, beta, eps=1e-5):
    batch_mean = np.mean(x, axis=0)
    batch_var = np.var(x, axis=0)
    x_norm = (x - batch_mean) / np.sqrt(batch_var + eps)
    out = gamma * x_norm + beta
    return out
```

BatchNorm的改变：
- 计算/更新均值
- 计算/更新方差
- 使用均值和方差将每个元素标准化

BN的特点：

沿着批量维度计算统计量（均值和方差）
对于CNN，通常应用于[N, C, H, W]的数据，统计维度为[N, H, W]
训练时使用小批量统计量，测试时使用整个训练集的运行统计量
优点：加速训练、允许更高学习率、减少对初始化的敏感性、训练更稳定、缓解梯度爆炸
缺点：对小批量size敏感，RNN中效果不佳


## layernorm

Layernorm指层归一化，Layer Normalization。

LN的特点：

沿着特征维度计算统计量，而不是批量维度
对于每个样本独立计算归一化参数
训练和测试阶段行为一致，不需要维护运行统计量
优点：适用于RNN和小批量场景，不依赖批量大小
缺点：在CNN中效果可能不如BN

## dropout
dropout通过随机将输入元素的置零来减少神经元之间的协同适应，强迫网络不能依赖任何单一神经元，从而防止过拟合
实现方法：通过随机掩码 + 缩放补偿
*在自然语言处理领域，随机掩码用于序列掩码操作，如BERT中的sequence mask。通过掩码部分输入，让模型学习如何从未被掩码的部分推断出被掩码的部分，从而提高模型的特征表示能力。*

## 初始化
模型初始化：
我们根据实际模型来使用torch.nn.init进行初始化。通常我们使用isinstance()来判断模块属于什么类型

参数初始化
参数初始化是训练模型的第一步，其核心作用在于：
- 避免输入数据经过多层传播之后出现梯度消失或梯度爆炸的问题
- 爆炸各层的输入和输出分布相对稳定
- 加快模型收敛速度
- 提高模型最终性能


1. 固定值初始化：用特定常数初始化（很少使用
同一神经元参数相同，训练过程中各神经元会保持一致的更新，等价于单个神经元。
2. 随机初始化：从概率分布中采样值作为初始参数，能有效打破对称性
- 均匀分布初始化
- 正态分布初始化
3. Xavier初始化
4. He初始化：考虑了ReLU会将一般输入置零的特性，专门针对ReLU激活函数设计



## 学习率衰减
在训练过程中逐步降低学习率

**先快后慢**
初期用较大的学习率快速逼近最优解
后期用较小的学习率精细调整，避免震荡或跳过最小值

常见学习率衰减方法：
1. 分段常数衰减：每个阶段使用不同的固定学习率
2. 指数衰减：学习率按指数函数下降
3. 自然指数衰减：按自然指数下降
4. 多项式衰减：使用多项式函数
5. 余弦退火：使用余弦函数模拟学习率变化，可结合周期性重启
6. 余弦衰减：类似余弦退火，但不重启

## earlystopping

epoch很大的时候，训练到后期，训练集的损失值越来越低，准确率越来越高，验证机的损失值却开始逐渐增高，并且越升越快。这就是典型的过拟合，模型把训练集的噪声都当成的规律，泛化能力也来越差。
使用早停能让模型在合适的时机停止，能避免模型过拟合，又能节约训练事件。

早停的核心原理是以验证集指标为监控依据，设置耐心值触发停止。
常见指标：分类任务优先选准确率、F1值、损失值；回归任务优先选MSE、MAE、验证集损失值。
设置耐心值：允许验证集连续多少轮不提升，然后触发早停
保存最优模型：持续监控验证集指标，一旦出现更优指标，就自动覆盖保存当前模型。所以最终早停出发时，我们手里保存的是整个训练过程中，验证集表现最好的那个模型，而非最后一轮模型
