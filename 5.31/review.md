## SVM
对于线性可分数据，SVM目标是在特征空间中找到一个最优超平面 $w^T x + b = 0$，使得两类样本间隔最大。

对于复杂问题，可能数据不能用线性函数分开，
那么SVM就可以使用kernel核函数把数据映射到更高维的空间（或者模拟映射），
让原本无法线性分开的数据变得可以线性分开。
这条线映射回原本的二维空间后可能表现为复杂的弯曲边界。

而支持向量是离边界最近的关键样本，也是决定最终模型的样本。
svm的核心也就是找到由支持向量确定的、间隔最大的分类边界。


适合svm处理的数据类型：
1. 数据量不是很大：几百至几万条不等
2. 特征比较多：几百几千个特征，因为svm本来就是在高维空间里工作的
3. 类别边界明显：通过核函数比较好分开


## KNN
K近邻算法属于监督学习的一种，核心思想是通过计算待分类样本与训练集中各个样本的距离，找到距离最近的K个样本，然后根据这K个样本的类别或值来预测待分类样本的类别或值。

KNN算法可以概括为以下步骤：
1. 计算距离：计算待分类样本与训练集中每个样本的距离（距离度量方法有欧氏距离，曼哈顿距离等）
2. 选择K个最近邻：根据计算出的距离，选择距离最近的K个样本
3. 投票或平均：对于分类问题，K个最近邻中出现次数最多的类别即为待分类样本的类比；对于回归问题，K个最近邻的值的平均值即为待分类样本的值。

KNN 中最重要的超参数就是K。
我们通常需要交叉验证来选择合适的K。如果K太小，结果容易受到噪声的干扰，如果K太大，容易把远处的样本也纳入范围内。多次尝试，选择预测表现最好的K。

```python
# 标准超参数调优

from sklearn.model_selection import cross_val_score
from sklearn.neighbors import KNeighborsClassifier

best_k = 0
best_score = 0

for k in range(1, 21):
    knn = KNeighborsClassifier(n_neighbors=k)

    score = cross_val_score(
        knn,
        X,
        y,
        cv=5
    ).mean()

    if score > best_score:
        best_score = score
        best_k = k

print(best_k, best_score)

```

KNN的训练速度非常快，但预测速度很慢，因为预测的时候几乎需要跟全部的样本算一遍距离。

KNN适合处理：
1. 类别具有明显聚集性的数据
2. 数据量不是太大的数据
3. 特征维度不太高的数据

## 朴素贝叶斯

朴素贝叶斯算法之所以叫朴素贝叶斯，是因为其假设各个特征相互独立。
朴素贝叶斯的工作流程概括如下：
首先训练数据，统计各类别出现的概率，然后统计每个特征在各个类别中出现的概率。
训练完成后，输入新样本，识别新样本中的特征，计算属于各个类别的概率，选择概率最大的类别。

朴素贝叶斯算法特别适合用于文本分类，用于分类简单分布的数据或者小数据集。




# KNN、SVM、朴素贝叶斯对比表

| 对比项 | KNN | SVM | 朴素贝叶斯 |
|----------|----------|----------|----------|
| 核心思想 | 看最近的 K 个邻居 | 寻找最大间隔分类边界 | 计算各类别概率 |
| 分类依据 | 邻居投票 | 决策边界 | 后验概率最大 |
| 训练过程 | 几乎没有训练 | 学习最优超平面 | 统计概率分布 |
| 预测方式 | 根据最近邻分类 | 根据边界分类 | 根据概率分类 |
| 是否依赖距离 | 是 | 否 | 否 |
| 是否需要标准化 | 非常需要 | 建议进行 | 一般不需要 |
| 训练速度 | 快 | 较慢 | 很快 |
| 预测速度 | 慢 | 快 | 快 |
| 适合小样本 | 可以 | 非常适合 | 适合 |
| 适合高维数据 | 不适合 | 非常适合 | 较适合 |
| 适合文本分类 | 一般 | 常用 | 非常常用 |
| 可解释性 | 中等 | 较低 | 较高 |
| 典型超参数 | K | C、kernel、gamma | alpha |
| 主要优点 | 简单直观 | 泛化能力强 | 训练速度快 |
| 主要缺点 | 高维效果差，预测慢 | 参数较多 | 特征独立假设较强 |

# 适用场景

## KNN

关键词：

- 距离
- 邻居
- 投票
- 相似样本

适合：

- 中小规模数据集
- 特征维度较低
- 同类样本聚集明显

典型应用：

- 鸢尾花分类
- 用户兴趣分类
- 简单推荐系统

记忆：

> 近朱者赤，近墨者黑

---

## SVM

关键词：

- 最大间隔
- 支持向量
- 核函数
- 高维数据

适合：

- 样本少
- 特征多
- 分类边界复杂

典型应用：

- 基因数据分类
- 医学诊断
- 文本分类

记忆：

> 小样本，高维数据优先考虑 SVM

## 朴素贝叶斯

关键词：

- 贝叶斯公式

适合：

- 文本分类
- 垃圾邮件识别
- 情感分析
- 新闻分类

典型应用：

- Spam/Ham 分类
- 评论情感分析
- 新闻自动分类
