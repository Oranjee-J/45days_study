# Pytorch

pytorch有以下几个概念：张量、自动求导、神经网络模块、优化器等。
张量是Pytorch的核心数组结构，支持多维数组，可以在CPU和GPU上进行加速计算
自动求导功能使得pytorch能轻松计算模型的梯度，以便进行反向传播和优化。


## 感知机/MLP

单层感知机是最简单的神经网络单元，模拟人脑神经元的工作方式，输入多个特征，加权求和，阈值判断，输出结果。但是单层感知机只能处理线性可分问题，不能解决非线性问题。
在单层感知机的输入输出层之间，加入一层或多层隐藏层，并在每层之后引入非线性激活函数，就得到了MLP。

全连接网络，也称为多层感知机。全连接网络表示每一层的每一个神经元都与相邻层的每一个神经元相连。
FCN非常适合处理结构化数据，如表格数据。
输入层仅接收原始数据和特征，不做任何计算，神经元数量=数据的特征维度，此层无激活函数。
隐藏层是MLP的核心层，可设置一层或多层，层数/神经元数是关键超参数。对输入层的特征加权求和后，通过激活函数输出，实现特征升维。
输出层输出模型最终的预测结果。


## 激活函数
激活函数是神经网络中的核心组件，通过引入非线性特性使网络能够学习复杂的关系，从而解决线性模型无法处理的问题。
某些激活函数还能帮助缓解梯度消失问题，从而提高训练效率。

比如ReLU()激活函数，它对输入张量逐元素执行修正线性单元运算，能有效解决梯度消失问题。
f(x) = max(0,x)
如果值大于0，保持不变，如果值小于等于0，输出0。
首先，ReLu是非线性关系，数据关系无法化简成简单的线性关系，模型能够学习复杂关系。其次，ReLU函数只需要判断，计算极快，这使得梯度更容易传播，训练深层网络更容易。
再次，在梯度传播里，ReLU的导数等于0(x<0)或者1(x>0)。这使得x大于0时，梯度乘1值不会变小，经过几个ReLU之后梯度大小保持不变


## 损失函数
所有的pytorch损失函数都是nn.Module的子类。

```python
# 实例化损失函数
criterion = nn.CrossEntropyLoss()
# 计算损失（预测值在前，真实值在后）
loss = criterion(predictions, targets)
# 反向传播
loss.backward()
```
不同损失函数对输入的形态要求不同。

CrossEntropyLoss: 输入原始数据，标签是整数类别索引
损失函数内部自动执行 Softmax + 对数 + 负号，无需手动对模型输出做 Softmax。
BCE Loss：输入是经过Sigmoid的概率处理后的0-1的概率值，标签是0/1浮点数
MSE Loss：输入任意实数，标签任意实数

## 反向传播
反向传播是根据损失函数计算梯度，然后以此更新网络的权重和偏置。
在此过程中，从输出层开始，使用损失函数逐层计算每个神经元的误差，然后利用这些误差和前一层的激活值来计算当前层权重的梯度。最后，使用这些梯度，通过梯度下降来更新网络的权重与偏置。

- 反向传播链式法则是否理解？
链式法则用来计算复合函数中的导数，复合函数的导数就是各部分倒数的乘积，链式法则使得我们能够逐层计算导数。
反向传播的链式法则能够计算损失函数对每一层参数的梯度，损失函数对输出的梯度与每一层的局部梯度相乘，逐层向后计算
